# 八王子 中古マンション価格分析

情報科学セミナーI（三木研）の分析レポート。国土交通省「不動産取引価格情報」を用いて、
八王子市とその沿線（京王線・JR中央線）の中古マンション価格が **どんな要因で決まり、
2024→2025でどう変わったか** を分析。テーマは「分析＝区別する／因果を明確にする」。

👉 公開レポート: https://hiraqoo1130.github.io/hachioji-mansion-analysis/

- 基本統計 → 交絡（構成効果）の切り分け → クラスタリング → 層別重回帰
- 結論：価格は依然「築古が主役」で決まり方はほぼ不変。見かけの変化は新築1K流入の構成交絡
